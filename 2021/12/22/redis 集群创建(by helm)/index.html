redis集群创建(by helm)

2021-12-22 | 运维

通过helm一键安装redis集群，在安装的时候，可以按需调整下几个基本参数，然后就可以一键部署。

helm fetch stable/redis

tar -zxvf redis.tgz

vi values.yaml

调整集群slave 个数以及是否需要开启哨兵（默认是关闭），true为开启false为关闭

#调整是否需要开启密码登录，默认是关闭

#调整master节点或slave节点需要开启nodeport访问，默认是没开启，service的类型是clusterIP

按需改好values.yaml后，就可以执行安装：

helm install redis-cluster –namespace default -f values.yaml stable/redis
